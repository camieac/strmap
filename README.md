# strmap

## Overview

This repository re-hosts the work of Per Ola Kristensson (http://pokristensson.com).
I have found the `strmap` library useful and thought I would make it available
from a public version controlled source.

This software has been redistributed here in accordance with [GNU LGPL](https://www.gnu.org/licenses/lgpl-3.0.txt).
The original source can be found at http://pokristensson.com/strmap.html.

## Examples
Examples usage of the strmap API can be found on [Per Ola Kristensson's website](http://pokristensson.com/strmap.html).
